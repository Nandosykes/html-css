# Estudos em HTML e CSS

Este repositório contém exercícios e páginas simples desenvolvidas para aprender os fundamentos de **HTML** e **CSS**.

## Objetivo
Compreender a estrutura de páginas web e conceitos básicos de estilização.

## Conteúdos abordados
- Estrutura HTML
- Tags semânticas
- Estilização com CSS
- Layouts simples

## Tecnologias utilizadas
- HTML5
- CSS3

## Como visualizar
Abra os arquivos `.html` diretamente no navegador.

## Observação
Projeto focado em aprendizado prático e construção de base para front-end.
